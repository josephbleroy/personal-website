## üé® Theme: Void-Raiders

The site uses a custom theme called **Void-Raiders**, designed to evoke a gritty, sci-fi aesthetic inspired by *Arc Raiders*.

**Key Features:**
- **Glitch Typography**: Animated, unstable text effects.
- **Void Background**: Subtle, shifting rainbow gradients.
- **Holo-Socials**: Interactive, pulsating social icons.
- **Retro-Tech Styling**: Terminal-like interfaces and high-contrast code blocks.

For more details on the theme, check out the [Theme README](themes/void-raiders/README.md).

## üöÄ Local Development

To run this site locally:

1.  **Clone the repository:**
    ```bash
    git clone --recurse-submodules https://github.com/josephbleroy/personal-website.git
    cd personal-website
    ```

2.  **Install Hugo:**
    Follow the [official installation guide](https://gohugo.io/installation/) for your operating system.

3.  **Run the development server:**
    ```bash
    hugo server -D
    ```

4.  **Access the site:**
    Open your browser and navigate to `http://localhost:1313`.
